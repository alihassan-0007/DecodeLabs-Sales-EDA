📊 DecodeLabs: Sales Data Exploratory Data Analysis (EDA)
📌 Project Overview
This project provides a comprehensive Exploratory Data Analysis (EDA) on the DecodeLabs sales dataset. The goal of this analysis was to transform raw, inconsistent data into actionable business insights by identifying sales trends, measuring product performance, and understanding customer acquisition channels.

🚀 Key Methodologies
To ensure the highest level of data accuracy, the following analytical processes were implemented:

Data Integrity Audit: Conducted a deep-dive audit to identify missing values, duplicate entries, and anomalies in the sales records.

Data Standardization: Standardized naming conventions across categories to ensure consistent grouping and meaningful analysis.

Strategic Imputation: Handled data gaps using logical imputation techniques, preserving the statistical integrity of the 1,200+ order dataset.

Business Intelligence: Utilized Pivot Tables and dynamic Pivot Charts to visualize performance metrics and revenue drivers.

📈 Key Insights
Revenue Concentration: Identified that a small percentage of products contribute to the majority of total revenue (Pareto Principle application).

Referral Source Performance: Analyzed which acquisition channels drive the highest conversion and revenue value.

Outlier Detection: Spotted high-value orders that deviate from the standard purchasing behavior, providing scope for targeted marketing.

🛠 Tools & Technologies
Excel: Data Cleaning, Pivot Tables, Pivot Charts, and Advanced Formula logic.

Data Visualization: Custom charts for distribution analysis and trend reporting.
